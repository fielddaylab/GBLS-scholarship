# Coded Journal Article Archive

Generated 7201 abstract-based coded article records from `journal-article-archive.jsonl`.

These records use the controlled values in
`0_human_sources/metadata_schema_and_lexicon.md`. Coding is inferred from
article titles and abstracts only; it is not a substitute for full-text
review. Empty Zotero and attachment identifiers were not invented. The
original abstract is preserved in each record's Summary section.
