## How to Move Papers in Zotero

### Option A: Manual Selection (for smaller batches)
1. In Zotero, go to IMLS Games and Libraries group
2. Use Advanced Search to find papers by key:
   - Click Gear icon → Advanced Search
   - Search for the Zotero key
   - Select the item
   - Right-click → Add to Collection → select destination collection

### Option B: Scripted via Zotero API
Use the provided Python script to move papers programmatically (requires API write permissions)

### Option C: BibTeX Export
Export a list of keys and use a tool to batch move them.

## Recommended Workflow

1. Create the two collections first
2. Start with the smaller batch (_ready_for_summary - 65 papers)
3. Then process _needs_pdf (128 papers)

Once organized, you can:
- Prioritize summarizing the 65 ready papers
- Track down PDFs for the 128 papers needing them
- Monitor progress as papers get summarized
