### Summary
# **Seay, J. A. (2018). Considering games and gamification in libraries & associated entities: The "G" word. *Against the Grain, 30*(4), 68-69. https://doi.org/10.7771/2380-176X.8311**

Seay's "The G Word" is a short professional column aimed at readers new to gamification. It is useful for the GBLS review because it explicitly distinguishes games from gamification, a distinction that is often blurred in practitioner discourse. The column begins with Ralph Koster's definition of a game as a rule system that creates a simplified model of some aspect of reality. Seay contrasts this with gamification, which does not necessarily involve playing a complete game. Instead, gamification applies game thinking, game dynamics, or game-like elements to a non-game context in order to engage people and solve problems.

The column draws heavily on Bohyun Kim's ACRL writing about library gamification. Seay emphasizes that gamification is not the same thing as library game night, board game collections, or simply letting patrons play. Its central function is motivation. Game mechanics such as points, progress tracking, levels, badges, leaderboards, quests, rewards, and competition can make ordinary activities feel more purposeful, visible, or compelling. Seay uses everyday examples such as fitness apps and Habitica to show how gamification turns routine activities into tracked, goal-oriented systems.

For libraries, the column argues that gamification fits a long-standing service problem: how to engage users with resources, programs, services, orientations, instruction, and professional development. Seay notes that academic libraries have applied gamification to student orientation and information literacy instruction. He also cites Kyle Felker's discussion of point scoring and the Lemon Tree game at the University of Huddersfield, where resource usage could be point-scored by department or discipline. These examples show how existing library data, systems, and user behaviors can become inputs for game-like engagement systems.

The article also stresses that gamification does not have to require expensive technology. Seay repeats Kim's point that a successful library instruction game can be run with pencil and paper, and he names summer reading badges as a simple example. This matters for GBLS because it separates gamification from both high-cost digital platforms and full game development. A library can gamify an activity through visible progress, goals, rewards, roles, and feedback even when no digital game is present.

One of the more distinctive parts of the column concerns gamification for librarian professional development and promotion. Drawing on Lindsay Cronk's LITA Blog writing, Seay imagines promotion as a mission with quests, battles, rewards, collaborators, research outputs, publications, and "power-ups." Although playful, this example widens the scope of GBLS-adjacent gamification beyond patron instruction. It suggests that library organizations can use gameful structures internally to frame professional learning, collaboration, and career advancement.

The column is also useful because it contains an implicit caution. Seay acknowledges that gamification can sound like behavior manipulation and describes the larger social trend in which apps, marketers, educators, and institutions incentivize everyday behavior. He prefers to frame library gamification as user-centered instructional engagement, but the ambivalence remains. For the GBLS review, that tension is important: gamification can motivate participation, but it also raises questions about whether points and badges support meaningful learning or merely nudge compliance.

For the GBLS project, this source should be treated as practitioner framing rather than evidence. It does not assess outcomes or give a detailed implementation case. Its value is conceptual vocabulary: games are bounded rule systems, while gamification is the application of game dynamics to non-game contexts. It helps the review avoid treating all game-related library work as equivalent. A board game night, a circulating game collection, an escape room, a gamified orientation, and a badge-based reading program may all belong under GBLS, but they operate through different mechanics, labor demands, and service goals.
