### Summary
# **Porter, M., Sewell, A., Bergstrom, G., & Costello, D. (2024). RTRT: International Games Month. *Google Docs*. https://docs.google.com/document/d/1cC7xc5nZrYRHbFUBOKRetIaRZBtpVP_fAbv4p6Q1Zjs/edit?usp=sharing&usp=embed_facebook**

This item is a one-page Round Table Round Table resource sheet for International Games Month, associated with Molly Porter, Amber Sewell, George Bergstrom, and Danielle Costello. The PDF is not an article, case study, assessment report, or practitioner essay. Instead, it gathers links for librarians and GameRT participants who want to learn about International Games Month and related ALA GameRT activity.

The document points readers to the International Games Month website; GameRT articles on International Games Month check-ins and International Games Week reports from 2019 and 2020; examples of past IGM events such as an itch.io jam, Eau Claire Public Library's games month, Miami Dade College and Valencia College guides, Brookline Public Library programming, and Nottingham City Libraries games clubs; and GameRT's Twitch policy. It also includes basic conduct rules for GameRT Twitch activity: kindness and respect, no hate speech, moderator removal authority, reporting harassment to a GameRT Twitch moderator, and the option to send a direct message for private concerns. The sheet closes with GameRT blog, Facebook, resource, website, and email links.

For the larger GBLS project, this resource is relevant as evidence of the professional infrastructure around library gaming. It shows that GameRT treats International Games Month as an ongoing program format with shared web infrastructure, examples, public-resource lists, and moderation policies for online gaming spaces. It also points toward practical topics that matter for library game services: event examples, public-facing guides, virtual programming, community conduct norms, and professional peer support.

However, the PDF itself should not be cited in `review.md` as substantive literature. It does not describe a specific library program in enough detail to analyze goals, staffing, participants, game formats, access barriers, outcomes, or assessment. It does not synthesize International Games Month practice, evaluate GameRT resources, or make an argument about game-based library services. The useful evidence likely lives in the linked IGM website, GameRT articles, and local library event pages, not in this brief link sheet.

This item is therefore best retained as explicitly excluded literature/resource material. It may be useful later as a starting point for locating examples of International Games Month programming or GameRT policy language, but adding it to the review would add a citation without adding much analysis. The current GBLS review already has stronger sources for gaming event history, library gaming programs, public-library TTRPG services, tabletop collections, virtual gaming, and professional rationales.
